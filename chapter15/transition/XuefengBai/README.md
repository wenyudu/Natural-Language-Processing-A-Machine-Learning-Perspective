# Local Transition-based Parser for Dependency Parsing

## Requirement
+ python >= 3.6
+ pytorch >= 1.0
+ tqdm


## How to run
`python run.py`

It will train a Transition-based Parser for **10** epoch, and finally get results as below:

| Name          | Value |
| ------------- | ----- |
| Training Loss | 0.029 |
| Dev UAS       | 88.69 |
| Test UAS      | 89.09 |
